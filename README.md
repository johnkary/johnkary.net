# Development

To build the site and allow Jekyll to serve it:

    $ jekyll serve --drafts

To build the site as-is to the build directory:

    $ jekyll build --future --drafts

# Deployment

1. Open Transmit
2. Connect to s3.amazonaws.com with Access Key ID and Secret
